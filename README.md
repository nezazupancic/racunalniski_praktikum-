# Računalniški praktikum
## Bližnjice na tipkovnici

+ *Kopiraj* označeno v odložišče: <kbd>Cmd</kbd>+<kbd>C</kbd> (**C**opy)
+ *Izreži* označeno v odložišče: <kbd>Cmd</kbd>+<kbd>X</kbd>
+ *Prilepi* vsebino odložišča: <kbd>Cmd</kbd>+<kbd>V</kbd>
+ *Shrani* <kbd>Cmd</kbd>+<kbd>S</kbd> 
+ *Išči* <kbd>Cmd</kbd>+<kbd>S</kbd>
+ *Izbere vse elemente* <kbd>Cmd</kbd>+<kbd>A</kbd>
